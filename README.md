## Run Locally  
Clone the project  

~~~bash  
  git clone https://github.com/ozgeahras/twitter-clone.git
~~~

Go to the project directory  

~~~bash  
  cd twitter-clone
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the app  

~~~bash  
npm run start
~~~
